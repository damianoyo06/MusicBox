# MusicBox
Music Box is an interactive visual programme that allows users to listen to multiple songs. While a song is playing, the box displays various information around it, including the key signature, release date, mood, and beats per minute. These details are dynamically extracted from a .csv file, demonstrating data handling and integration within the application.

At the front of the box, there is a standard music player interface showing the song title, artist, total duration, and current playback time, along with play, next, and previous buttons. The top half of the box features a waveform that dynamically visualises the song’s volume.

This interactive application was developed in Processing4 using Java.
