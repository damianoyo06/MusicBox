🎵 MusicBox

MusicBox is an interactive visual application that allows users to listen to multiple songs while exploring detailed information about each track in real time.

Features
Play, pause, skip to next/previous songs
Displays song metadata:
- Key signature
- Release date
- Mood
- Beats per minute (BPM)
Dynamic waveform visualisation that reacts to the song’s volume
Clean music player interface showing:
- Song title
- Artist
- Total duration
- Current playback time

All song data is dynamically loaded from a .csv file, demonstrating effective data handling and integration within the application.

Built With
- Processing 4
- Java

How to Run
1. Download the project as a ZIP file
2. Extract the ZIP to your desired location
3. Open Processing 4
4. Navigate to and open the Assignment_5 folder
6. Run the project by clicking the Run button (▶️)
   
📁 Project Structure
Assignment_5/ – Main Processing sketch
.csv file – Contains song data and metadata
Additional assets – Audio files and visual components

📌 Notes
Make sure all files (especially the .csv and audio files) remain in their original locations relative to the sketch
Processing must have permission to access audio files on your system
